# Raven Documentation

This folder is the root for repository documentation.

## Structure

- `docs/design/`
  - Architecture and design documentation.
  - Current canonical architecture document:
	- `docs/design/Raven.Architecture.Design.md`

## Authoring Guidance

- Put architecture and technical design direction under `docs/design`.
- Put future user-facing docs (guides, setup, operations, usage) directly under `docs/` or user-doc subfolders as they are introduced.
- Avoid creating multiple parallel architecture plan files; update the canonical design document instead.
